# AI Sentiment Analysis for Social Media

## Overview
This project uses Machine Learning and Natural Language Processing to classify text as Positive or Negative.

## Technologies Used
- Python
- Flask
- Scikit-Learn
- TF-IDF
- Naive Bayes

## Features
- Real-time sentiment prediction
- Professional web interface
- Confidence score display
- Trained on social media sentiment data

## How to Run

pip install pandas scikit-learn flask joblib

python train.py

python app.py

Open:
http://127.0.0.1:5000
